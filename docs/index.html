¿Cómo es posible?

El sistema de Certificados de Ahorro Energético (CAE) permite dar valor al ahorro energético generado por una actuación de eficiencia.

  • Evaluación técnica: analizamos la viabilidad del proyecto.
  • Mejora del confort: menos pérdidas térmicas en la vivienda.
  • Gestión integral: acompañamos el proceso técnico y documental.
